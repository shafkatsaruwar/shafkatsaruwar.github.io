National Park Matcher

Full-stack web application for discovering US National Parks through location-based search and trip planning

Python Flask HTML5/CSS3 Jinja2 REST APIs

Overview

Collaborative capstone project building a Flask-based web application for discovering US National Parks. Users can search 400+ parks by state or activity and get personalized driving directions.

Key Features

  • Multi-criteria search supporting state-based and activity-based filtering
  • Real-time data integration with National Park Service API
  • Location-based Google Maps directions from user's starting point
  • Dynamic park detail pages with images and descriptions
  • Support for all 50 states and US territories

My Contributions

As the frontend developer, I was responsible for the user interface design and implementation:

  • Built main landing page with dual search functionality
  • Developed complete CSS styling system for consistent design
  • Created park listing and result pages
  • Collaborated on Flask routing integration

Photo Sorter

Python CLI tool organizing images by EXIF metadata with automated audit reporting

Python Pillow (PIL) EXIF Parsing CSV CLI

Overview

Command-line utility that automatically organizes photos into folders based on their EXIF metadata (date taken, camera model) and generates detailed CSV reports for auditing.

Key Features

  • Automated sorting by date or camera model
  • EXIF metadata extraction from hundreds of images
  • CSV report generation for quality assurance
  • Batch processing of large photo libraries
  • Graceful error handling for corrupted files

Impact

Reduced manual photo organization time from 30 minutes to under 1 minute, saving approximately 90% of manual effort through automated EXIF parsing and intelligent file organization.

Payroll Manager for HR

Bash automation system for payroll processing with tax calculations and searchable database

Bash Linux Utilities grep/awk Automation

Overview

Interactive command-line tool that automates payroll calculations for HR departments, featuring time tracking, progressive tax calculations, and a searchable employee database.

Key Features

  • Automated weekly timesheet processing
  • Progressive tax bracket logic (5%-15% based on hours worked)
  • Searchable employee database with fuzzy matching
  • Interactive CLI menu system with input validation
  • Monthly pay projections and tax withholding

Impact

Eliminated manual computation errors and reduced payroll processing time by an estimated 70% through automation of calculations, tax logic, and employee record management.
